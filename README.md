# Empleos SIMO
El objetivo de esta aplicación web es observar y comparar los diferentes empleos que se encuentran en la aplicación SIMO para empleos publicos colombianos a partir de un JS Object "data" que normalmente colocariamos al inicio del codigo fuente en el archivo app.js. En este caso los datos usados son de algunos empleos del concurso de ascenso de la DIAN 2025.
